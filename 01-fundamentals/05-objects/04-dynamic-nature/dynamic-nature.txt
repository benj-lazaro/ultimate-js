Dynamic Nature Of Objects
-------------------------
- Objects in JavaScript are dynamic meaning once created
    - It is possible to add new or remove existing
        - Properties
        - Methods

    - To dynamically remove a property and/or method from an object
        - Use the "delete" operator
        - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/delete)

    Syntax:
    
    delete object.property
    delete object[property]
