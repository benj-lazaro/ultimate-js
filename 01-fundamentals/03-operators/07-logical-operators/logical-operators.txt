Logical Operators
-----------------
- Logical operators are used to make decision based on multiple conditions
    - Typically used with Boolean values
    - Returns a Boolean value
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_operators#logical_operators)


- Logical AND (&&)
    - Returns true if BOTH operands evaluates to true
    - Otherwise, returns false


- Logical OR (||)
    - Returns true if EITHER ONE of the operands evaluates to true
    - If both operands are false, it returns false


- Logical NOT (!)
    - Returns false if the single operand evaluates to true
    - Returns true if the single operand evaluates to false
