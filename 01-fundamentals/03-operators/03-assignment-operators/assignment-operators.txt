Assignment Operators
--------------------
- It is used to assign a value to a variable or property
    - The assignment expression itself has a value w/c is the assigned value
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Assignment)


- It is an operator that assigns a value to its left operand
    - Based on the value of its right operand
    - Reference (https://devdoc.net/web/developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Assignment_Operators.html)



Syntax:

x = y
