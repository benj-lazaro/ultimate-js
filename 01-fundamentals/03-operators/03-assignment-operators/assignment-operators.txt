Assignment Operator
-------------------
- An Assignment operator is used to assign a value to a variable or an object's property
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Assignment)

    - It is an operator that assigns a value located at the right-hand side of the operand
        - To the operand located on its left-hand side

            let x = 10;
            x = x + 1;


Syntax:

x = y
