Objects
-------
- It is a reference type of data
    - It represents an object in real life


- To define an object
    - It uses an Object Literal Notation (syntax) 
    - Within the pair of { }
        - Define the key-value pairs (properies) 


- Two ways of accessing an object property
    - Dot Notation
        - The name of the object
        - Followed by a "."
        - Then the name of the property to access

    - Bracket Notation
        - The name of the object
        - Followed by a pair of "[ ]"
        - Inside the pair is the name of the property to access
