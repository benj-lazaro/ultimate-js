Functions
---------
- A Function (like a variable) is one of the fundamental buildng blocks in JavaScript
  - It is a set of statements that performs a specific task and/or returns a value

  - Like the Array, it is a Reference data type

  - It can optionally accepts a parameters of varying data types
    - A parameter (param) is a variable defined at the time a function is declared
    - Maximum number of arguments (values stored in parameters) varies in different JavaScript Engines
  
  - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/function)


Syntax:

function name(param0) {
  statements
}

function name(param0, param1) {
  statements
}

function name(param0, param1, /* â€¦, */ paramN) {
  statements
}
