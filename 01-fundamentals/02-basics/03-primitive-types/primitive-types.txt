Primitive Types
---------------
- JavaScript has two (2) categories of data types:
    - Primitive a.k.a Value Type
        - string    = sequence of characters
        - number    = integer, float
        - boolean   = true / false
        - undefined = default value of an uninitialized variable
        - null      = used for clearing the assigned value in a variable
        - symbol    = introduced in ES6

    - Reference Type
        - array
        - object
        - function
