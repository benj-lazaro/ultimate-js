Filtering an Array
------------------
- The .filter() is a method of the Array instances
    - Creates a shallow copy of a portion of a given array
    - Filtered down to just the elements from the given array 
    - That pass the test implemented by the provided callback function
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter)


Syntax:

filter(callbackFn)
filter(callbackFn, thisArg)
