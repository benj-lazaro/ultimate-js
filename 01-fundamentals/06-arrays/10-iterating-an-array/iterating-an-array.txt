Iterating an Array
------------------
- An alternative to using for..of loop when iterating elements of an array 
    - Use the .forEach() method
        - Offers the ability to access the index of the elements
        - Offers the ability to use a callback function on each element


- The .forEach() is a method of the Array instances
    - It executes a provided function once for each array element
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach)


Syntax:

forEach(callbackFn)
forEach(callbackFn, thisArg)
